# 第06回 確率

- 平均，分散，共分散を NumPy で計算する。
- 一様分布と正規分布から標本を生成し，ヒストグラムで確認する。
- 標本数と乱数 seed が推定値に与える影響を確認する。
- 正規分布の平均を最尤推定の観点から探す。

## 解説
- 平均はデータの中心，分散はばらつき，共分散は2つの変数が一緒に増減する傾向を表す。
- 確率分布は，値がどの範囲にどの程度現れやすいかを表す。有限個の標本から見える分布は，真の分布そのものではなく近似である。
- 一様分布は範囲内を同じ確率で取り，正規分布は平均の近くに値が集まりやすい。
- 最尤推定は，観測されたデータが最も起こりやすくなるパラメータを選ぶ考え方である。

## 演習
### 基礎レベル（7問）
1. `scripts/`，`outputs/session06/`，`outputs/figures/` を作成し，`scripts/session06_probability.py` で `data = np.array([2, 4, 5, 9, 10])` の平均，分散，標準偏差を計算する。
2. `x = [1, 2, 3, 4, 5]`，`y = [2, 3, 5, 7, 11]` の共分散を計算し，散布図を保存する。
3. `np.random.default_rng(0)` を使い，一様分布 `U(0, 1)` から 1000 個の標本を生成してヒストグラムを保存する。
4. 正規分布 `N(0, 1)` から 1000 個の標本を生成してヒストグラムを保存し，一様分布と形を比較する。
5. 標本数を 10，100，1000 に変え，標本平均と標本分散がどう変わるか表にする。
6. `N(mu, 1)` の `mu` を `-1` から `1` まで動かし，観測データに対する log likelihood を計算する。
7. `outputs/session06/session06_report.md` に，平均，分散，共分散，ヒストグラム，最尤推定で選ばれた `mu` をまとめる。

### 発展レベル（7問）
1. `ddof=0` と `ddof=1` の分散を比較し，不偏分散が何を補正しているかを書く。
2. 共分散が正，負，0 に近い3種類の synthetic data を作り，散布図と共分散を比較する。
3. 正規分布の標準偏差を 0.5，1.0，2.0 に変え，ヒストグラムの広がりを比較する。
4. log likelihood を likelihood の積で直接計算した場合と比較し，log を使う理由を説明する。
5. 一様分布と正規分布の標本から，平均 ± 1 標準偏差の範囲に入る割合を計算する。
6. seed を変えたときの標本平均のばらつきを20回分調べ，結果を図にする。
7. 「標本から推定した値」と「真の分布のパラメータ」の違いを，今回の実験に即して説明する。
