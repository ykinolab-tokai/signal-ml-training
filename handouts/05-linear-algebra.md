# 第05回 線形代数

- ベクトルと行列の和，内積，行列積を NumPy で計算する。
- ノルム，射影，線形変換の意味を図と数値で確認する。
- 逆行列と連立一次方程式の関係を確認する。
- shape を見ながら，計算可能な行列演算を判断する。

## 解説
- ベクトルは数値を並べたもの，行列はベクトルを変換するものとして扱える。NumPy では shape を確認しながら計算することが重要である。
- 内積は2つのベクトルの向きの近さを測る計算として見られる。ノルムはベクトルの長さを表す。
- 射影は，ある方向に沿った成分を取り出す操作である。残差が射影先の方向と直交することを確認すると意味が分かりやすい。
- 逆行列は，線形変換を元に戻す行列である。ただし，すべての行列に逆行列があるわけではない。

## 演習
### 基礎レベル（7問）
1. `scripts/`，`outputs/session05/`，`outputs/figures/` を作成し，`scripts/session05_linear_algebra.py` で2つのベクトル `u = [1, 2]`，`v = [3, -1]` の和，差，スカラー倍を計算する。
2. `u` と `v` の内積，ノルム，なす角の cos を計算する。
3. 行列 `A = [[2, 1], [1, 3]]` とベクトル `x = [1, -1]` について，`A @ x` を計算し，shape を表示する。
4. 2つの行列 `A`，`B = [[0, 1], [2, 1]]` について，`A + B`，`A @ B`，`B @ A` を比較する。
5. `v` を `u` の方向へ射影し，射影ベクトルと残差ベクトルの内積が 0 に近いことを確認する。
6. 格子点を作り，回転，拡大縮小，せん断の3種類の線形変換を図にする。
7. `A @ x = b` を `np.linalg.solve` で解き，`np.linalg.inv(A) @ b` と一致するか確認する。

### 発展レベル（7問）
1. `A @ B` と `B @ A` が一般には一致しないことを，数値例と文章で説明する。
2. 特異行列 `S = [[1, 2], [2, 4]]` に逆行列が存在しないことを確認し，determinant と rank を表示する。
3. 射影先のベクトルを変えたとき，射影と残差がどう変わるか図にする。
4. `np.linalg.cond` を使い，条件数の大きい行列では解が不安定になりやすいことを小さな例で確認する。
5. `np.linalg.solve` と `np.linalg.inv` を使う実装を比較し，連立方程式を解くときにどちらを優先すべきかを書く。
6. 3次元ベクトルでも内積，ノルム，射影が同じ式で書けることを確認する。
7. 線形変換の図を見て，元の図形の平行性，長さ，角度が保たれるかを変換ごとに整理する。
