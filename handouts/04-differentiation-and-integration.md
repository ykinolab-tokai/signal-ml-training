# 第04回 微分と積分

- 傾きと数値微分を，式と計算の両方で確認する。
- 偏微分と勾配を NumPy で近似する。
- 関数の極値を，グラフと勾配の観点から確認する。
- 区分求積法，定積分，数値積分を実装する。

## 解説
- 微分は，入力を少し動かしたときの出力の変化率を表す。数値微分では，小さな `h` を使って差分から傾きを近似する。
- 多変数関数では，1つの変数だけを動かした変化率を偏微分と呼ぶ。各変数についての偏微分を並べたものが勾配である。
- 関数の極値では，勾配が 0 に近くなることが多い。ただし，数値計算では刻み幅や探索範囲によって近似結果が変わる。
- 積分は，小さな区間の面積を足し合わせる操作として近似できる。区分求積法や台形公式を実装すると，定積分を数値で扱える。

## 演習
### 基礎レベル（7問）
1. `scripts/`，`outputs/session04/`，`outputs/figures/` を作成し，`scripts/session04_differential_integral.py` で `f(x) = x**2 + 3*x + 1` を定義する。
2. 前進差分と中心差分で `f'(1)` を近似し，解析的な値 `2*x + 3` と比較する。
3. `h = 1e-1, 1e-3, 1e-5, 1e-7` で数値微分を行い，刻み幅による誤差の変化を表にする。
4. `x` の範囲を `[-4, 2]` とし，`f(x)` と数値微分の値を同じ図に描く。
5. `q(x, y) = x**2 + x*y + y**2` について，点 `(1, 2)` での偏微分を中心差分で近似する。
6. `f(x) = (x - 2)**2 + 1` に対して，勾配降下法を10 step 実行し，`x` が極小点へ近づく様子を記録する。
7. 区分求積法で `∫_0^1 x^2 dx` を近似し，分割数を 10，100，1000 に変えて結果を比較する。

### 発展レベル（7問）
1. `sin(x)` の `0` から `pi` までの定積分を，区分求積法と台形公式で近似し，真値 2 と比較する。
2. `h` を小さくしすぎたときに数値微分の誤差が必ず減るとは限らないことを確認し，理由を書く。
3. `q(x, y)` の等高線を描き，点 `(1, 2)` における勾配ベクトルを重ねる。
4. 勾配降下法の learning rate を 0.01，0.1，1.0 に変え，収束の違いを比較する。
5. `f(x) = x**4 - 3*x**2 + 1` を grid search で調べ，極小点の候補を探す。
6. 台形公式の分割数と誤差の関係を log scale の図にする。
7. 微分と積分が，後のノイズ除去や信号復元でどのように現れそうか，自分の言葉で3点書く。
